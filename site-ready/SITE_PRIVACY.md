# Skill Cleaner Site Privacy

## Safe To Show Publicly

- product name;
- public promise;
- six workflow stages;
- safety boundaries;
- output contract;
- public GitHub link;
- release ZIP checksum;
- generated product visuals.

## Do Not Show Publicly

- private project memory;
- user credentials;
- browser/session data;
- Drive mirrors;
- raw local dirty skill folders;
- private test outputs containing local paths unless intentionally sanitized.

## Public Safety Line

```text
Skill Cleaner creates a safe output library first. It does not permanently delete source files, install unknown packages, or execute unknown skills.
```
