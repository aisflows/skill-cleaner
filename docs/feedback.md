# Feedback

Skill Cleaner is useful only if it cleans real messy skill folders, not just fixtures.

Use GitHub Issues for feedback:

```text
https://github.com/aisflows/skill-cleaner/issues/new/choose
```

## Best Feedback

Open a feedback issue if you tried Skill Cleaner and can answer at least one question:

- What kind of folder did you run it on?
- Did it create `skill-library-cleaner-output/clean-library/`?
- Did `FINAL_LIBRARY_MAP.md` explain the keep/archive/block decisions?
- Which risky files were blocked correctly?
- Which useful files were missed or routed badly?
- What would make the next run easier?

## Do Not Share

Do not include:

- secrets, tokens, cookies, browser profiles, `.env` files;
- private customer data;
- private prompts or paid skill content;
- screenshots that expose local paths or account data.

## Short Feedback Format

```text
Input shape:
Dry-run or approved write-run:
Output created:
FINAL_LIBRARY_MAP.md useful? yes/no
One thing that worked:
One thing that failed or confused me:
```
