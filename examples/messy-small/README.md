# Messy Small Fixture

These files are intentionally stored as `EXAMPLE_SKILL.md.txt`.

Reason: public marketplaces and agent-skill scanners often crawl every file named `SKILL.md`. The fixture includes duplicate and risky sample skills, so publishing them as real `SKILL.md` files would make the repository look like it contains active skills that are only test inputs.

To run this fixture locally, copy the `raw/` folder to a temporary test folder and rename each `EXAMPLE_SKILL.md.txt` to `SKILL.md` inside that temporary copy. Do not rename the files in the public repository package.
